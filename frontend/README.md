# LoreGraph Frontend

React-based frontend for LoreGraph — an AI-powered MCU canon checker and knowledge graph explorer.

## Setup

```bash
npm install
npm start
```

The app runs on [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Features

- **Canon Verification** — Upload story PDFs and get an AI contradiction score against MCU canon
- **Knowledge Graph** — Visual Neo4j graph of MCU characters, events, locations, and timelines
- **Story Management** — Create and manage your story collection
- **Report Modal** — Detailed breakdown of canon conflicts with recommendations

## Tech Stack

- React 19
- React Router v7
- NeoVis.js (Neo4j graph visualization)
- Axios
- Redux Toolkit
